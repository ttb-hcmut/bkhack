# BKHack

A social news website oriented towards computer science, for HCMUT

## How to develop

Start nix shell

```sh
nix-shell
```

Assume you've successfully entered nix shell

Prepare

```sh
pnpm install
pnpm run init
```

Run dev server

```sh
pnpm dev
```

Voila!
